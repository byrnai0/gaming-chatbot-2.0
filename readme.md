# Chatbot 2.0 ()

## Overview

This project is a chatbot application designed to interact with the RAWG video game database. The bot is currently under development.

## Current Status

- The chatbot has been created, but its API integration is pending and will be implemented soon.
- The RAWG database has been established and connected via its API.
- Basic information can be retrieved from the database. Additional features for fetching specific details are still in progress.
